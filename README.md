# Harsha Skills Marketplace

Custom Claude Code / Cowork plugin marketplace.

## Plugins

### harsha-custom-skills
11 custom skills for research, security, mobile, workflow protection, and more.

**Slash commands:**
- `/harsha-custom-skills:deep-research`
- `/harsha-custom-skills:mckinsey-research`
- `/harsha-custom-skills:security-audit`
- `/harsha-custom-skills:codebase-documenter`
- `/harsha-custom-skills:kissht-release-notes`
- `/harsha-custom-skills:workflow-guardian`
- `/harsha-custom-skills:mobile-mastery`
- `/harsha-custom-skills:tech-stack`
- `/harsha-custom-skills:parallel-builder`
- `/harsha-custom-skills:super-pipeline`
- `/harsha-custom-skills:web-brainstorm`

## Installation

```bash
claude marketplace add harshavardhanbailur-kissht/harsha-skills-marketplace
claude plugin install harsha-custom-skills@harsha-skills-marketplace --scope user
```
