# LAP-2052 — LSQ : Dropdown Changes | property_occupancy_status | property_document

The idea is to make the list of the Property Document (type of ownership document) and Property Type (property occupancy status) to be same across LSQ.

## Leadgen

* At Property Documents Task
    * Update the 'Primary Property Document/Type of ownership document' dropdowns as per the list attached
    * Upload document functionality should be provided against each dropdown with proper name
* At Loan and Property
    * Update the 'Property occupancy status / Property type' dropdown as per the list attached. And same should be reflected in the detail / mortgage detail tab
* Values from leadgen should be autofetched in the LOS as per the current implementation

## LOS

* For Fresh Loan | Normal with Waiver or Normal without Waiver
    * At Credit PD form for Primary applicant (both at edit and first time initiation, at all credit stages)
        * 'Type of ownership document / Property Document' and 'Property occupancy status' dropdowns has to be updated
    * Property Technical form (both at edit and first time initiation, Stages: Third Party Trigger Pending, Final Sanction Pending, Final Sanction Approval Pending)
        * 'Type of ownership document / Property Document' and 'Property occupancy status' dropdowns has to be updated
    * Recommendation form (both at edit and first time initiation, Stages: Final Sanction Pending, Final Sanction Approval Pending)
        * 'Type of ownership document / Property Document' and 'Property occupancy status' dropdowns has to be updated

* For Fresh Loan | SARAL
    * Review Property Technical (both at edit and first time initiation, Stages: Final Sanction Pending, Final Sanction Approval Pending)
        * 'Type of ownership document / Property Document' and 'Property occupancy status' dropdowns has to be updated

* For BT + Topup
    * BT Category (both at edit and first time initiation, Stages: CPA Verified, In-principle Approval Pending)
        * 'Type of ownership document / Property Document' and 'Property occupancy status' dropdowns has to be updated
    * Any other touchpoint should be freezed once the category is defined at credit PD, Property technical and recommendation

* Document replacement functionality for 'Type of ownership document / Property Document' should be provided at Document Upload and cleanup tab at Post sanction stage with proper name as per the dropdown list for new and current values. Also selection of the dropdown values will remain freezed.
* At all the touch points where we are fetching the document for 'Type of ownership document / Property Document' field, on changing the value upload functionality should be given with the proper name as per the list.

## QA Scenarios

* Check if Sales PD tasks are getting generated with the new and updated dropdown values
* Check if any touchpoint is being missed in the ticket for above two fields
* Check if the values are proper in the finbox input parameter
* Check if the dropdown values are same as the sheet link
