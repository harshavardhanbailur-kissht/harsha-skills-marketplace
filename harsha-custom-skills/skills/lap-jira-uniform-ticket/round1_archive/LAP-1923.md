**Attendance Module** -

* Create a section to track attendance of the Sales Agents, this attendance module will enable SMs to mark their attendance where they will get these dispositions for marking attendance -
    * `[TO BE PICKED UP LATER]` (Present / Absent / Official holiday)
    * Feature to enable checkin and checkout for the user along with clicking selfie.
    * Once the user marks the attendance as Present → They will be asked to upload live selfie.
    * `[TO BE PICKED UP LATER]` The form that will be used by the user to mark attendance will only ask the disposition of Present/Absent/ Official Holiday once in a day.
    * `[TO BE PICKED UP LATER]` The SM will use the same form 4 times a day, where they will upload - selfie 2, 3 and 4.
    * We will need to store the daily data to be consumed for tracking purposes → Report to be created and made available to the managers of sales agents.
    * The next day, disposition will be taken again and the user repeats the same journey.
    * The day they will not mark their attendance - it be automatically taken as absent.
    * Also all weekdays this will be done except for Sundays.
