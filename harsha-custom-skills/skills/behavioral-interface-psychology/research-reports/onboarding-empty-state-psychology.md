# Onboarding & Empty State Psychology Research

## Executive Summary

This report synthesizes findings on user onboarding, empty states, and progressive feature discovery from Appcues, Chameleon, Nielsen Norman Group, Amplitude, and academic literature.

---

## 1. First-Run Experience Psychology

### First Impression Timing

**The 50ms Aesthetic Judgment Window:**
- Users form aesthetic judgments in as little as **50 milliseconds**
- Visual complexity and prototypicality affect ratings within 50ms, even within **17ms**

**The 2.6 Second Value Proposition Window:**
- Takes approximately **2.6 seconds** for user's eyes to land on the area that most influences first impression

**The Halo Effect:**
- Aesthetically pleasing websites are remembered as more usable than they actually were

### Cognitive Load During Onboarding

- Working memory: **4-7 pieces of information** at once
- Progressive disclosure improves: learnability, efficiency, and error rate

### Tutorial Completion Research (NN/Group)

**Key Findings:**
- Tutorials don't make users faster or more successful
- Users who read tutorials perceive tasks as **more difficult**
- No advantage in success rates or task-completion times
- Users frequently skip tutorials

**Recommendation:** Avoid app onboarding whenever possible; make UI more usable instead.

### Time-to-Value Metrics

| Metric | Value |
|--------|-------|
| Users churning without value in first week | 90% |
| Users not engaging within 3 days churn rate | 90% |
| Active users lost within 3 days (bad onboarding) | 75% |
| Lost within first month | Up to 90% |
| Best SaaS time-to-value | Under 5 minutes |
| First 5 minutes optimization impact | 50% increase in LTV |

## 2. Onboarding Pattern Effectiveness

### Product Tours Completion Rates (Chameleon 2025)

| Steps | Completion Rate |
|-------|-----------------|
| 4 steps | 40.5% |
| 5 steps | 21% (~50% drop) |

**Key Finding:** Top 1% completion rate tours don't exceed 5 steps.

### Interactive vs Video vs Documentation

| Method | Effectiveness |
|--------|--------------|
| Interactive | 50% higher activation rates than static |
| Interactive content engagement | 13 minutes vs 8.5 minutes (static) |
| Video comprehension increase | 80% (Forrester) |

### Sample Data vs Empty Start

- **Guided setup** increases conversion by **30-40%**
- **Empty states guidance** reduces confusion by **28%**
- Sample content should be clearly labeled and dismissible

## 3. Empty State Design Psychology

### Psychological Impact

- Blank screens trigger anxiety and uncertainty
- Creates "micro-moment of uncertainty" that increases friction

### Key Design Principles

Every empty state should include:
1. Explanation of why the state is empty
2. Motivation explaining why users should take action
3. Clear next steps or calls-to-action

### Critical Timeframe

- Users decide which apps to stop using within first **3-7 days**

## 4. Progressive Feature Discovery

### Feature Adoption Curves

- Adoption follows **S-curve** pattern
- Power user features might only hit **5% adoption** (still valuable)

### Just-in-Time Feature Introduction

- JITAIs show **moderate to large effect sizes** on outcomes
- Contextual help outperforms upfront tutorials by **2-3x**

### Upsell Timing Psychology

- **Success-triggered** upgrade prompts: **40% higher conversion** than time-based
- **>2 upgrade prompts per session**: **3x more likely to abandon**

### Conversion Benchmarks

| Type | Rate |
|------|------|
| Freemium | 2.6% |
| Free trial (opt-in) | 18.2% |
| Free trial (opt-out) | 48.8% |

## 5. Habit Formation

### Hook Model (Nir Eyal)

1. **Trigger:** External or internal prompts
2. **Action:** Behavior in anticipation of reward
3. **Variable Reward:** Tribe, Hunt, or Self rewards
4. **Investment:** Action improving future value

### Habit Loop Timing

- Formation starts within approximately **2 months** on average
- Significant variability: **4 days to nearly a year**
- **Debunked:** The "21 days" myth

### Tiny Habits Research

- Leaders starting with minimal viable habits were **2.7x more likely** to maintain long-term habits

### The 7% Retention Rule (Amplitude)

- When **7%** of original cohort returns on day seven, product has demonstrated early value
- Strong correlation between early activation and 3-month retention

### Retention Correlation

- **25% increase in activation** = **34% rise in MRR** over 12 months

## 6. Onboarding Friction

### Sign-up Form Optimization

| Metric | Value |
|--------|-------|
| Average form abandonment | 67% |
| Eliminating one field impact | +50% conversions |
| Reducing 11 fields to 4 | +120% conversions |
| Optimal form fields | 3 |
| Mandatory phone number abandonment | 39% |
| Optional phone number abandonment | 4% |
| Multi-step vs single-page forms | Up to +300% |

### Social Login Benefits

- Registration rate increase: Up to **50%**
- Sign-up improvement with SSO: **20-40%**
- One-click sign-up improvement: **8.2 percentage points**

### Verification Step Abandonment

- **64%** drop off during typical SaaS signup
- **Best Practice:** Grant immediate access, verify later

### Personalization Impact

- Personalized paths increase completion by **35%**
- Personalized users are **30% more engaged**

## 7. User Segmentation in Onboarding

### Beginner vs Expert Paths

- Hotjar's skill-level checklists led to **26% increase** in installations
- Use AI/ML for adaptive learning paths

### Goal-Based Onboarding

- AdRoll's goal quizzes: **35% increase** in feature usage
- **86%** more likely to stick with products offering educational content

## 8. Gamification in Onboarding

### Progress Indicators Psychology

- **Zeigarnik Effect:** More likely to complete unfinished tasks
- **Endowed Progress Effect:** Pre-filling steps motivates completion
- Dopamine release when earning badges/points

### Case Studies

- **Talana:** 40% increase in activation with gamified checklists
- **LinkedIn:** 55% increase in profile completion with progress bars

### Checklist Best Practices

- Include **3-5 key tasks** to reach activation point

## Do's and Don'ts

### Do's

1. Get users to value in under 5 minutes
2. Limit product tours to 4-5 steps
3. Use contextual, just-in-time help
4. Personalize based on user goals
5. Use checklists with progress indicators
6. Keep signup forms to 3 fields maximum

### Don'ts

1. Don't rely on static tutorials
2. Don't show >2 upgrade prompts per session
3. Don't require mandatory phone numbers
4. Don't create truly empty states
5. Don't require email verification before showing value
