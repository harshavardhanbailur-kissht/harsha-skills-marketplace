# Research Statistics & Business Case

> Comprehensive data supporting mobile UX investment decisions

## Mobile Traffic & Commerce Statistics

### Global Mobile Usage (2024-2025)

| Metric | Value | Trend |
|--------|-------|-------|
| Mobile share of web traffic | 58-64% | ↑ Increasing |
| Mobile time in apps vs browser | 88% apps, 12% browser | Stable |
| Average mobile session | 72 seconds | ↓ Decreasing |
| Daily smartphone pickups | 96 times/day average | ↑ Increasing |
| Mobile commerce value | $2+ trillion annually | ↑ 15% YoY |

### Mobile Commerce Performance Gap

| Channel | Conversion Rate | Cart Abandonment |
|---------|----------------|------------------|
| Desktop | 3.09-3.9% | 67-69% |
| Tablet | 3.11-3.49% | 70-72% |
| **Mobile Web** | 1.8-2.96% | 75.5-79.6% |
| **Mobile App** | 4.6-6% (157% higher) | 65-68% |

**Key Insight:** Mobile apps convert **157% higher** than mobile web. App users view **4.2× more products** per session and spend **64% more time** engaged.

### Cart Abandonment Causes (Mobile)

| Reason | Percentage |
|--------|------------|
| Extra costs (shipping, tax, fees) | 48% |
| Required account creation | 26% |
| Complicated checkout process | 22% |
| Couldn't calculate total cost | 21% |
| Site errors/crashes | 18% |
| Didn't trust site with credit card | 17% |
| Delivery too slow | 16% |
| Return policy concerns | 12% |

**Recoverable value:** Baymard Institute estimates **$260 billion** in lost orders could be recovered through better checkout UX.

### Page Speed Conversion Correlation

| Load Time | Conversion Rate | Bounce Rate |
|-----------|----------------|-------------|
| 0-2 seconds | 1.9%+ | ~25% |
| 2-3 seconds | 1.5% | ~32% |
| 3-4 seconds | 1.2% | ~38% |
| 4-5 seconds | <1% | ~45% |
| 5+ seconds | <0.6% | ~58% |

**Key Finding:** Every **1-second delay** costs **7-20% in conversions**.

### Current CWV Pass Rates (Chrome UX Report)

| Metric | Mobile Pass Rate | Desktop Pass Rate |
|--------|-----------------|-------------------|
| LCP | ~50% | ~64% |
| INP | ~65% | ~97% |
| CLS | ~78% | ~80% |
| All Three | ~41% | ~53% |

**Note:** INP (Interaction to Next Paint) replaced FID in March 2024. The gap between mobile and desktop INP pass rates reveals hidden responsiveness issues on mobile.

## Touch Interaction Statistics

### One-Handed Usage Patterns

| Usage Pattern | Percentage |
|--------------|------------|
| One-handed (thumb only) | 49% |
| Cradled (one hand, other finger) | 36% |
| Two-handed (thumbs) | 15% |

### Thumb Zone Engagement

| Zone | Screen Area | User Engagement |
|------|-------------|-----------------|
| Natural (easy reach) | Bottom 40% | Highest engagement |
| Reach (moderate) | Middle 33% | Moderate engagement |
| Stretch (hard) | Top 27% | -43% engagement vs natural zone |

### Touch Target Statistics

| Target Size | Miss-tap Rate | User Satisfaction |
|-------------|---------------|-------------------|
| <32px | >30% | Poor |
| 32-40px | 15-25% | Below average |
| 40-44px | 5-10% | Average |
| 44-48px | 2-5% | Good |
| 48px+ | <2% | Excellent |

**Key Finding:** Touch targets smaller than 44px cause **25%+ higher miss-tap rates**, leading to frustration and abandonment.
