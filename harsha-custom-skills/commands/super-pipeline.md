# /super-pipeline — Super Skill Pipeline

Unified thinking-planning-execution pipeline that intelligently routes tasks through up to 3 stages: deep thinking, orchestration, and parallel execution. A complexity gate picks the right depth.

## Invocation

```
/super-pipeline Build a complete loan application tracking system
/super-pipeline Think deeply about this architecture, then build it in parallel
/super-pipeline [upload a complex project brief]
```

## Workflow

Load the `super-pipeline` skill. The complexity gate determines which stages are needed.
