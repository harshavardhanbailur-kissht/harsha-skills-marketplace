# /ultimate-debugger — Ultimate Debugger

Unified debugging and code quality system. Inherits from gas-debugger and performance-debugger with added safe fixing methodology and cross-framework bug patterns.

## Invocation

```
/ultimate-debugger React component re-renders infinitely on state change
/ultimate-debugger API returns 500 on mobile but works on desktop
/ultimate-debugger Memory leak in Three.js scene after navigation
```

## Workflow

Load the `ultimate-debugger` skill and execute the 5-phase debugging pipeline: context/triage, reproduction/measurement, root cause analysis, fix design with verification, and improvement/commitment.
